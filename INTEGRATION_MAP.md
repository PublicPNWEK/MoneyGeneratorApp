# Integration Map

Screen | Entry points | Integration actions
---|---|---
Overview | Quick links to Billing, Settings, Finance | Navigation only (fast access to billing center, Plaid setup, CRM webhooks)
Subscriptions | Billing center | Start subscription checkout, launch PayPal approval, confirm/cancel provider subscription, refresh catalog, update payment method token, annual plan upsell, add-ons purchase
Settings → Billing | Shortcut to billing center | Uses same subscription actions; shows subscription status
Settings → Connected Accounts (Plaid) | Plaid link token issuance | Issue link token, mark account linked via token exchange; view read-only accounts and sync state
Settings → Integrations (CRM webhooks) | Webhook configuration summary | Configure/read webhook destinations for CRM updates (HubSpot/Salesforce/custom HTTPS)
Finance | Read-only Plaid dashboard | Refresh read-only balances/accounts and transaction feed; no editing or payments
